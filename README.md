# ⏳ Shutdown Timer

![Shutdown Timer UI](https://img.shields.io/badge/App-Shutdown_Timer-2ea44f?style=for-the-badge)
![Platform](https://img.shields.io/badge/Platform-Windows-blue?style=for-the-badge)

**Shutdown Timer** is a simple, sleek, and lightweight standalone desktop application built to schedule automatic shutdowns for your Windows PC. With a modern interface and bilingual support (Arabic & English), it makes setting a sleep timer effortless.

---

## ✨ الميزات (Features)
- 🖥️ **إيقاف التشغيل التلقائي (Auto Shutdown):** حدد عدد الساعات، الدقائق، والثواني وسيقوم البرنامج بإطفاء الجهاز تلقائياً بدقة.
- ⚡ **اختصارات سريعة (Quick Presets):** أزرار جاهزة لضبط التايمر بضغطة واحدة (15 دقيقة، 30 دقيقة، ساعة، ساعتين، و 3 ساعات).
- 🌐 **دعم اللغات (Bilingual):** يدعم اللغتين العربية والإنجليزية بشكل كامل، مع إمكانية التبديل بينهما من الواجهة.
- 🎨 **تصميم عصري (Modern UI):** واجهة مستخدم جذابة وداكنة (Dark Mode) مع حلقة دائرية (Ring Progress) توضح الوقت المتبقي.
- 🔽 **العمل في الخلفية (System Tray):** يمكنك تصغير البرنامج (Minimize) ليعمل بهدوء في الـ System Tray بجوار الساعة دون أخذ مساحة من الشاشة.
- 🚀 **نسخة محمولة (Portable):** لا يحتاج لأي تثبيت (No Installation required)، فقط حمل البرنامج وشغّله.

---

## 🚀 كيفية الاستخدام (How to use)
1. قم بتحميل الملف التنفيذي **[`ShutdownTimer.exe`](https://github.com/Ahmed-Atef-Tech/ShutdownTimer-Release/raw/main/ShutdownTimer.exe)** من هذا المستودع.
2. قم بتشغيل البرنامج بضغطة مزدوجة.
3. اختر الوقت الذي تريده (ساعات : دقائق : ثواني) أو اختر من الأزرار السريعة.
4. اضغط على زر **"تشغيل التايمر" (Start Timer)**.
5. *لإلغاء الإطفاء:* يمكنك فتح البرنامج مرة أخرى والضغط على **"إلغاء" (Cancel)** في أي وقت قبل انتهاء العداد.
