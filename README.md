# Shutdown Timer

This repository contains the standalone executable version (`.exe`) of the Shutdown Timer application. 

## Features
- Easily schedule your computer to shut down after a specific time.
- Simple and user-friendly interface.
- Lightweight and requires no installation, just run the executable.

## Download
You can download the application directly from the files above:
- **`ShutdownTimer.exe`**

## How to use
1. Download the `ShutdownTimer.exe` file.
2. Double-click to run the application.
3. Set the desired time and click the button to start the timer.
